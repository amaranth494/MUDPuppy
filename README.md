# MUDPuppy

A MUD (Multi-User Dungeon) game client and framework.

## Overview

MUDPuppy is a modern MUD client built with web technologies, designed to provide a seamless gaming experience for MUD enthusiasts.

## Technology Stack

- **Frontend:** Node.js
- **Backend:** Go
- **Database:** PostgreSQL
- **Cache:** Redis
