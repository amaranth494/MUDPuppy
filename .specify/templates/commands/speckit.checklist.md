## Usage

/speckit.checklist [artifact to validate]

## Description

Generate quality checklists to validate requirements completeness, clarity, and consistency. This command:

1. Creates validation checklists from templates
2. Validates specifications, plans, or tasks
3. Reports pass/fail status for each criterion
4. Identifies items requiring attention

## Example

/speckit.checklist Validate the user authentication specification
